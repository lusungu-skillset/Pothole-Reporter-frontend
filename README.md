## 🧠 System Overview

- **Frontend:** Next.js + Tailwind CSS  
- **Backend:** Node.js + Express  
- **Database:** PostgreSQL  
- **Maps:** Leaflet + OpenStreetMap  
- **Auth:** JWT-based authentication
