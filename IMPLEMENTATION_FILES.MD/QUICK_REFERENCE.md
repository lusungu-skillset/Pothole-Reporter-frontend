# Admin Dashboard - Quick Reference Card

## 🎯 Three Main Views

### 1. Dashboard Tab (📊)
**Purpose**: Overview of key metrics and insights

**What You See**:
- Total reports count
- Pending / In Progress / Resolved counts with percentages
- Severity distribution chart (Critical, High, Medium, Low)
- District distribution chart
- Status progress bar
- Quick insights and recommendations

**When to Use**: Daily check-in, morning briefing

---

### 2. All Potholes Tab (📋)
**Purpose**: Manage individual potholes with detailed controls

**What You Can Do**:
- **Search**: Type road name or description
- **Filter**: By status, severity, district, date range
- **Sort**: By date, severity, or status
- **View Details**: Click "View" button for full information
- **Update Status**: Use dropdown in table
- **Open in Maps**: Click "Maps" button for coordinates
- **Assign Staff**: Use staff dropdown in details modal
- **Add Notes**: Add comments and updates
- **Bulk Operations**: Select multiple and mark as resolved or export

**Table Columns**:
| Column | Function |
|--------|----------|
| ☑ | Select checkbox |
| Road Name | Pothole location |
| District | Administrative area |
| Location | Street/area description |
| Severity | Color badge (🔴🟠🟡🟢) |
| Status | Color badge (⏳🔧✅) |
| Date Reported | Report date |
| Actions | View, Maps, Status, Delete |

**When to Use**: Daily work, managing reports, assigning tasks

---

### 3. Analytics Tab (📈)
**Purpose**: Identify patterns, trends, and problem areas

**What You See**:
- **Response Time**: Average hours to respond
- **Hotspots**: 5 highest concentration areas with coordinates
- **Problem Roads**: Top 5 roads needing attention
- **7-Day Trend**: Report count by day (bar chart)
- **Response Distribution**: How fast issues are being handled

**When to Use**: Weekly planning, resource allocation, trend analysis

---

## 🎮 Quick Action Guide

### Urgent: Report is Critical 🔴
1. Go to **All Potholes** tab
2. Filter: Severity = "Critical"
3. Click "View" on the report
4. Change status to "In Progress"
5. Assign to staff member
6. Add note with urgency details
7. Click "Maps" to see exact location

### Routine: Update Progress 📝
1. Go to **All Potholes** tab
2. Filter: Status = "In Progress"
3. Click on pothole row
4. Add note about progress
5. Update status when complete

### Report: Generate Monthly Data 📊
1. Go to **All Potholes** tab
2. Set date filter: This month
3. Click "Export All CSV"
4. Save file for records
5. Use data for reports

### Analysis: Find Problem Areas 🎯
1. Go to **Analytics** tab
2. Review "Most Problematic Roads"
3. Check "Geographic Hotspots"
4. Note top areas needing resources
5. Plan preventive maintenance

### Bulk Update: Mark Multiple Done ✅
1. Go to **All Potholes** tab
2. Filter: Status = "In Progress"
3. Check boxes for completed work
4. Bulk actions bar appears
5. Click "Mark as Resolved"
6. Confirm action

## 📊 Color Legend

### Status Badges
- 🟠 **Pending** (Orange) - Waiting to start
- 🔵 **In Progress** (Blue) - Being worked on
- 🟢 **Resolved** (Green) - Completed

### Severity Badges
- 🔴 **Critical** (Red) - Urgent, danger zone
- 🟠 **High** (Orange) - Serious, needs soon
- 🟡 **Medium** (Yellow) - Important
- 🟢 **Low** (Green) - Minor

## 🆘 Common Issues

| Issue | Solution |
|-------|----------|
| No data showing | Refresh button or wait 5 seconds |
| Filter not working | Check exact value (case-sensitive) |
| Maps not opening | Check location has coordinates |
| Export shows nothing | Make sure potholes match filters |
| Notes disappeared | Check browser storage wasn't cleared |
| Slow response | Backend may be overloaded, try refresh |

## 💾 Data Backup

Notes are saved locally in your browser. To backup:
1. Export CSV regularly
2. Keep downloaded CSVs for records
3. Notes auto-save to browser

## 👥 Staff Assignment Workflow

1. Go to **All Potholes** → **View** detail
2. Scroll to "Assign to Staff" section
3. Select staff member from dropdown
4. Click "Assign" button
5. See confirmation message
6. Staff member is now assigned
7. Add note about the assignment
